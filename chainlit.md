# The Apothecary's Shopping Apprentice

This is the **chapter 12** pydantic-ai agent — the same buying capability as the
chapter 11 MCP server, but now wrapped in a program with its own model, persona,
and reasoning loop.

Make sure the backend is running (`make backend-dev`), then just ask:

- *"What can I afford right now?"*
- *"Stock me up for healing potions."*
- *"Buy 10 moonpetal."*

See `workshop/12-pydantic-ai-agent.md` for the full exercise.
